Introduction to Web Programming

Tony Hetrick: Instructor, Software Engineering Department

This course will introduce students to the technologies and techniques of web programming where students will learn how to develop interactive web pages.  Students will learn programming for both client-side and server-side technologies.  This includes basic web technologies (HTML5, CSS), client-side technologies (JavaScript, AJAX), and server-side technologies (PHP,MySQL).
